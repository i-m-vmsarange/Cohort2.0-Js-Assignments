# this keyword

- value of this keyword changes depending upon where it is written
- for example it is different in event handler,function, object, method, class

# different location value

- global me this ki value window hoti hai
- function me this ki value window hoti hai
- object me normal function me this ki value object hoti hai
- object me arrow function me this ki value window hoti hai
- this ki value event handler me khud jispe event laga hai vo element hota hai
- class me humesha this ki value blank object hoti hai
- arrow function apne this ki value humesh apne parent se leta hai
